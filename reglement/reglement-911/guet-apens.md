# GUET-APENS

<details>

<summary></summary>

Les guet-apens peuvent être organisés uniquement dans les cadres suivants :

<mark style="color:purple;">**Rendez-vous pour transaction illégale**</mark> (armes, drogues, etc.),

<mark style="color:purple;">**Interpellation d’une personne recherchée**</mark>

<mark style="color:purple;">**Collecte d’informations**</mark>

﻿﻿﻿﻿﻿﻿

En participant à un guet-apens contre un groupe, vous acceptez le risque d’une <mark style="color:purple;">**Mort-RP selon le contexte.**</mark>

L’objectif prioritaire reste la création de <mark style="color:purple;">**trames RP sur le long terme**</mark>, en privilégiant les scènes, le jeu et le fairplay plutôt que l’arrestation immédiate.

</details>
